# Simple Interest Calculator

This project calculates simple interest using the formula:

Simple Interest = (P × R × T) / 100

## Inputs
- P = Principal amount
- R = Rate of interest
- T = Time (in years)

## Example
If:
P = 1000  
R = 5%  
T = 2 years  

Simple Interest = (1000 × 5 × 2) / 100 = 100

## Usage
You can use this formula in any programming language or calculator.
